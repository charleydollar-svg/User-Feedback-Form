##USER FEEDBACK FORM

Description: 
   -- A feedback page where users can input their Name, Email, (Both up to a limit of 50 characters) and up to 500 characters worth of written feedback. Error pop up messages occur when either of the test boxes are empty when feedback is submitted. A previous feedback section is included for other website viewers to read past feedback. 

Technologies:
    --This script handles character counting, tooltips, click events, and form submission for a feedback form.
    --It listens for input events to update character counts, mouseover events to show tooltips, and click events to log background clicks while preventing them when clicking inside the form. It also validates form submission and displays feedback entries.

How to run:
    --Download or clone this repository
    --Makes sure that you have an "index.html", "style.css" and "cp_1.js" files in the same folder.
    --open the "index.html" in your web browser

What I learned from building this project
    --I learned how to build an interactive feedback form that had text boxes with character limits, a previous feedback list fom submitted entries, and I edited the page using user-friendly colors for professionalism.
